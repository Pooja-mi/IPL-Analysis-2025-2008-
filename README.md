# IPL Data Analysis (2008–2025) – Interactive Dashboard using Power BI

## Project Objective  
The Indian Premier League (IPL) wants to create a comprehensive performance analysis dashboard for all seasons from 2008 to 2025. So that, cricket analysts and fans can understand team performance, player achievements, and season-wise trends to gain meaningful insights from historical data.

---

## Dataset Used  
- <a href="https://github.com/Pooja-mi/IPL-Analysis-2025-2008-/blob/main/ipl_matches_data.csv">Matches Dataset</a>
- <a href="https://github.com/Pooja-mi/IPL-Analysis-2025-2008-/blob/main/teams_data.csv">Team Dataset</a>
- <a href="https://github.com/Pooja-mi/IPL-Analysis-2025-2008-/blob/main/players-data-updated.csv">Player Dataset</a>

---

## Questions (KPIs)  
- Who won the IPL season (Champion) for the selected year?  
- Which team was the Runner-up?  
- How many total matches were played in the selected season?  
- How many total teams participated?  
- What is the total number of sixes hit?  
- What is the total number of fours hit?  
- How many centuries were scored?  
- How many half-centuries were scored?  
- Which player won the Orange Cap (most runs)?  
- Which player won the Purple Cap (most wickets)?  
- Which player hit the most sixes?  
- Which player hit the most fours?  
- What does the points table look like for the selected season?  
- How many total venues were used?  

- Dashboard Interaction <a href="(https://github.com/Pooja-mi/IPL-Analysis-2025-2008-/blob/main/IPL_Dashboard.png)">View Dashboard</a>

## Process  
- Collected IPL datasets from reliable sources.  
- Imported datasets into Power BI.  
- Cleaned and transformed data using Power Query.  
- Verified data for missing values and inconsistencies.  
- Created relationships between match, team, and player tables.  
- Built DAX measures for KPIs such as Total Sixes, Total Fours, Total Matches, and Total Teams.  
- Designed cards, tables, and charts for season-wise analysis.  
- Added slicers for season selection and dynamic filtering.  
- Created an interactive and visually appealing dashboard layout.

## Project Insight  
- Royal Challengers Bangalore emerged as the Champion in the selected season.  
- Punjab Kings finished as Runner-up.  
- The Orange Cap winner scored the highest total runs in the season.  
- The Purple Cap winner took the highest number of wickets.  
- Teams with consistent wins dominated the points table.  
- Batting performance shows increasing trends in total sixes and fours over the years.

---

## Final Conclusion  
The IPL dashboard provides a complete season-wise and player-wise analysis of tournament performance. It helps in identifying top-performing teams and players across seasons. This project demonstrates how historical sports data can be transformed into meaningful business intelligence using Power BI. Such dashboards can be useful for analysts, coaches, and fans to explore trends and patterns interactively.
